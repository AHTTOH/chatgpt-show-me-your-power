# ChatGPT 사용법을 뼛속까지 (안똔AI) - 슬라이드 대본/포인트

## 1. 타이틀
- 메인: ChatGPT 사용법을 뼛속까지 (안똔AI)
- 부제: Apps / 개발자설정 / MCP / 자동화 / 실전 데모
- 오늘의 전체 로드맵을 한 줄로 예고

## 2. 오늘 얻어갈 것 5가지
- 화면 구조 이해
- Apps와 모델 역할 분리
- 개발자 설정 핵심
- MCP 연결/안전 규칙
- 실전 데모 자동화 흐름

## 3. ChatGPT 화면 구조 한 장 정리
- 모델/툴/앱/파일/프로젝트를 한 화면에 매핑
- 각각의 역할과 연결 포인트 설명
- 실제 화면에서 어디인지 지목

## 4. Apps: 앱이 하는 일 vs 모델이 하는 일
- 앱은 UI/권한/데이터 연결
- 모델은 사고/생성/요약
- Canva 예시로 역할 분리

## 5. 개발자 설정: 뭘 켜면 뭐가 달라질까
- 도구 허용과 자동 호출
- 파일 처리 옵션
- 메모리/프로젝트 설정
- 권한/보안 옵션

## 6. MCP: 개념 + 안전하게 쓰는 룰
- MCP = 도구 호출 표준
- 최소 권한 원칙
- 민감 데이터 처리 기준
- 로그/감사로 추적

## 7. 실전 데모 1: 문서/표/리서치 3-step
- 1) 자료 수집/요약
- 2) 표/목차로 구조화
- 3) 인사이트 도출
- 반복 가능한 템플릿 강조

## 8. 실전 데모 2: 자동화 시나리오
- 조건/트리거 정의
- 출력 형태 결정
- 실패 대비와 재시도
- 작은 자동화부터 확장

## 9. Codex란?
- ChatGPT 채팅과 차이
- 프로젝트/파일 중심 작업
- 실행/검증까지 이어짐

## 10. Codex로 산출물 만들기: 끝까지
- 코드/문서/슬라이드 자동화
- 버전 관리 흐름
- 오늘 실습 소개

## 11. 체크리스트: 실패 vs 성공 지시문
- 실패 사례: 모호/형식 누락
- 성공 사례: 목표/제약/검수 기준
- 프롬프트 품질이 결과 좌우
